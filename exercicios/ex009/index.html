Listas

listas ordenadas

são aquelas em que a ordem dos itens são muito importantes. seria tipo uma receita passo-passo de um bolo.Para fazer uma lista ordenada usarei a tag ol de ordered list junto com a tag li de list item.

  1. item 1
  2. item 2

eu posso tambem usar um parametro type dentro da tag ol para configura o marcador da lista , que tem por padrao o tipo 1 que cria uma lista numeriaca como no exemplo acima, mas a outros como A, que cria uma lista alfabetica maiuscula , a que cria uma lista alfabética minuscula, I que cria uma lista com algorismos romanos maiusculos e i que cria uma lista com algorismos romanos minusculos

exemplo 2

  1. item 1
  2. item 2
  3. item 3
  4. item 4

eu posso mistura uma lista dentro da outra.
exemplo 3

  1. item 1
  2. item 2
    1. item 2.1
    2. item 2.2
  3. item 3
  4. item 4

alem disso consigo usar o paramentro start dentro da tag ol para indicar onde a contagem do marcador irá começar.
exemplo 4

  1. item A
  2. item B

lista não ordenada

é o inverso da ordered list, onde a ordem dos itens não afetará o significado da lista, aqui usarei a tag ul de unordered list junto com a mesma tag li para identificar os itens da lista . eu posso usar nas listas nao ordenadas o parâmetro type so que com tipos diferente para as não ordenadas, disc é o padrao e fará a marcação com uma bola totalmente preta, circle fará uma bola com as bordas pretas e sem preenchimento interno, e por ultimo square que fará uma quadrado pintado de preto.

  • abacate
  • limão

  • açai

lista de definições

é tipo um dicionário, onde não a marcações como nas ordered list e unordered list, e possui um termo e sua defição. para fazer as listas de definições usarei primeiro a tag dl (defition list), apos ela usarei a tag dt(definition term) e depois a tag dd(definition description) onde irei colocar a definição do termo que coloquei na tag dt

HIPERLINKS

Para eu colocar um hiperlink basta usar a tag a de ancora. Toda vez que eu estiver usando um site e ao apertar em algum canto ele me direciona à outro site ou outra pagina do próprio site, eu estou interagindo com um hiperlink.

O principal parâmetro dessa tag é o href que é onde eu irei colocar a url do site a qual eu queira redirecionar. outro parâmetro importante é o hreflang que indica o idioma do site e já informa ao navegador se vai precisar traduzir

exemplo 1
clique aqui para assistir meu video no youtube

por padrão, sempre que o usuário abre um hiperlink no site, será aberto o site ou a pagina do link na mesma pagina onde eu estava, isso é o padrão, mas á parâmetros para mudar isso.

_blank
usado para abrir o link em outra janela em branco
_self
é o padrão onde o link irá ser aberto na janela atual ou frame atual
_top
desfaz todos os frames e abri em um destino com o navegador completo

o uso de flames está em desuso, os principais são os _blank e _self, para colocar isso utiliza-se o parâmetro target

E se o link for meu? E se for de outra pessoa?

existe um parâmetro chamado rel que é sobre a natureza do destino do link.

next
indica que o link é para a próxima parte do documento atual
prev
indica que o link é para uma parte anterior à parte do documento atual
author
indica que o link é para o site do autor do material
external
indica que o link é para outro site que não faz parte do site atual
nofollow
indica que é um link que não se sabe a procedência.

quando for link internos do meu proprio site eu não preciso digitar a url completa, basta apertar crtl+esp. já quando for links externos se faz necessário a url completa no parâmetro href.
